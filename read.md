# Final Project — SQL & Python (DURHAM)

This repository contains the final data analytics project combining SQL and Python. It includes the notebooks and scripts used for analysis, SQL queries, and visualization.

## Structure

- `CODE/` : Python notebooks and scripts
- `LICENSE` : project license
- `read.md` : this file (instructions and notes)

## Requirements

- Python 3.8 or newer
- pip
- (Optional) Virtual environment: `python -m venv .venv`

Install common dependencies:

```
pip install pandas numpy sqlalchemy matplotlib jupyter
```

If a `requirements.txt` file is present:

```
pip install -r requirements.txt
```

## Running

- Start Jupyter Lab/Notebook:

```
jupyter lab
```

- Open the notebooks from the `CODE/` folder.
